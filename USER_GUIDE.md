# 📱 Инструкция по Использованию Domi Sushi

## 🏠 Домашняя Страница (/)

На главной странице вы найдёте:
- **Навигационное меню** - ссылки на все основные разделы
- **Герой-секция** - привлекательное изображение и информация о ресторане
- **Бүгінгі арнайы ұсыныс** - сегодняшние специальные предложения
- **Біздің фирмалық тағамдарымыз** - специальные блюда ресторана
- **Ас ішу іс-шаралары** - информация о событиях и праздниках
- **Біздің тарихымыз** - история ресторана

**Кнопки Действия**:
- 🔑 **Кіру / Тіркелу** - Вход или Регистрация
- 📋 **МӘЗІРГЕ ӨТУ** - Перейти на страницу меню
- 🎯 **Брондау** - Бронировать столик

---

## 🍣 Страница Меню (/menu/)

На странице меню доступны:
- **Загрузка PDF** - скачать полное меню или меню напитков
- **Генератор Заказов** - случайный выбор блюд с помощью кубика
- **Викторина** - тест о знаниях суши (опционально)

### Как скачать меню:
1. ✅ Отметьте нужные файлы (Толық мәзір или Сусындар мәзірі)
2. 🖱️ Нажмите кнопку "Таңдалған файл(дар)ды жүктеу"
3. 📥 Файл автоматически скачается

### Генератор Заказов:
1. 🎲 Установите количество сторон кубика (4-100)
2. 🍽️ Выберите количество блюд (1-10)
3. 🎯 Нажмите "Сгенерировать заказ"
4. 📝 Система предложит случайный набор блюд

## 🔐 Вход и Регистрация (/auth/)

### Вход (Кіру):
1. 👤 Введите логин (имя пользователя)
2. 🔑 Введите пароль
3. ✅ Нажмите кнопку "Кіру"

**Результат**: 
- ✅ Успешный вход → Перенаправление на домашнюю страницу
- ❌ Неверные данные → Сообщение об ошибке

### Регистрация (Тіркелу):
1. 👤 Введите логин (имя пользователя)
2. 📧 Введите email адрес
3. 🔑 Введите пароль или нажмите "Генерациялау" для создания сильного пароля
4. 👁️ Нажмите "Көрсету" для проверки пароля
5. ✅ Нажмите "Тіркелу"

**Требования к паролю**:
- ✓ Минимум 8 символов
- ✓ Содержит строчные буквы (a-z)
- ✓ Содержит прописные буквы (A-Z)
- ✓ Содержит цифры (0-9)
- ✓ Содержит специальные символы (!@#$% и т.д.)

**Результат**:
- ✅ Успешная регистрация → Автоматический вход в систему

## 👤 Профиль Пользователя (/profile/)

После входа в систему вы сможете:
- 📋 Просмотреть свои данные:
  - Логин
  - Email
  - Дата регистрации
  - Телефон (если указан)
  - Мекенжай (адрес, если указан)

**Опции на странице профиля**:
- 🏠 **← Басты бетке** - Вернуться на домашнюю страницу
- 🚪 **Шығу** - Выход из системы

## 🔓 Выход (/logout/)

Для выхода из аккаунта:
1. Нажмите кнопку **Шығу** в навигационном меню или на странице профиля
2. Вас перенаправит на домашнюю страницу
3. Статус аутентификации будет очищен

## ⚠️ Важная Информация

1. **Безопасность пароля**: Используйте сильные пароли с разными символами
2. **Email**: Укажите правильный email при регистрации
3. **Логин**: Логин должен быть уникальным (не может быть два пользователя с одинаковым)
4. **Сессия**: Время сессии истекает после закрытия браузера
5. **Забыли пароль**: Данная версия не имеет функции восстановления пароля

## 🆘 Решение Проблем

### "Неверное имя пользователя или пароль"
- ✓ Проверьте правильность написания логина
- ✓ Убедитесь в правильности пароля (регистр букв важен!)
- ✓ Проверьте, что зарегистрировались перед входом

### "Это имя пользователя уже занято"
- ✓ Используйте другое имя пользователя
- ✓ Or log in if you already have an account (Кіру)

### "Пароль тым әлсіз" (Слабый пароль)
- ✓ Используйте пароль с минимум 8 символов
- ✓ Добавьте заглавные буквы
- ✓ Добавьте цифры
- ✓ Добавьте специальные символы (!@#$%^&*)
- 💡 Используйте кнопку "Генерациялау" для автоматического создания
