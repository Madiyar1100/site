## 🚀 Как Использовать

### 1. Запуск сервера:
```bash
cd c:\Users\user\Desktop\lab2_2_new
python manage.py runserver
```
Сервер запустится на http://127.0.0.1:8000/

### 2. Вход в админ-панель:
- URL: http://127.0.0.1:8000/admin/
- Логин: admin
- Пароль: admin123

### 3. Тестирование:
- Посетите http://127.0.0.1:8000/ - главная страница
- Нажмите "Кіру / Тіркелу" для входа или регистрации
- Создайте новый аккаунт или используйте существующий
- Посмотрите профиль в /profile/
